## 1. 사용자 역할

| 역할 | 설명 | 식별 방식 | 접근 가능 화면 |
|------|------|-----------|---------------|
| **Host (교사)** | 세트지 제작, 배포, 리포트 조회 | 로그인 (UUID) | 대시보드 전체 |
| **Member (학생 계정)** | QR/링크로 참여. 학습 데이터 누적 | 로그인 (UUID) | 과제 목록 + 플레이 + 결과 |
| **Guest (비회원 학생)** | 가입 없이 QR/링크로 참여 | 이메일 + 닉네임 | 플레이 + 결과만 |

**핵심 규칙**
- Host는 대시보드에만 직접 접근. Member/Guest는 QR/링크로만 진입.
- 게스트가 동일 이메일+닉네임으로 재접속 시 동일 인물로 이어받기 (기기/브라우저 무관).
- 게스트 이메일로 회원가입 시 기존 게스트 데이터가 Member에게 자동 연결.

## 2. 핵심 플로우

### 2-1. 라이브 모드

```
[Host] 세트지 제작 → 배포 설정 → QR 생성
                                    ↓
[학생] QR 스캔 → 닉네임 입력 (Guest) / 로그인 (Member) → 대기
                                    ↓
[Host] 게임 시작 버튼 클릭
                                    ↓
[학생] 문항 풀기 (카운트다운 + 즉시 피드백)
                                    ↓
[Host] 실시간 컨트롤 (일시정지/건너뛰기/힌트 공개)
                                    ↓
[전체] 결과/랭킹 화면
                                    ↓
[Host] 리포트 분석 (문항별/학생별)
```

### 2-2. 과제 모드

```
[Host] 세트지 제작 → 과제 배포 (기간 설정) → 이메일 알림/링크 공유
                                    ↓
[학생] 기간 내 자율 접속 → 문항 풀기 (이어하기 가능)
                                    ↓
[학생] 최종 제출 (1회, 확인 팝업)
                                    ↓
[Host] 제출 현황 확인 → 마감 처리
                                    ↓
[Host] 리포트 분석
```

### 2-3. Host 대시보드 (Sitemap)

```
홈 대시보드
├── 내 자료
│   ├── 세트지 목록 → 에디터 (자료 만들기)
│   └── 배포 설정
│         ├── [라이브] QR/링크 배포 → 대기화면 → 게임 시작
│         └── [과제] 기간 설정 + 알림 발송
├── 그룹 관리
│   ├── 그룹 목록 (생성/삭제)
│   └── 그룹 상세 (멤버 초대/제거, 게스트 링크)
└── 히스토리 & 리포트
      ├── 배포 히스토리
      ├── [과제] 제출 현황 대시보드
      └── 리포트 (문항별 / 학생별)
```

## 3. 배포 모드 비교

| 항목 | 라이브 (Live) | 과제 (Assignment) |
|------|--------------|-------------------|
| 진행 방식 | 교사 실시간 진행 | 학생이 기간 내 자율 진행 |
| 입장 방법 | QR/링크 (즉시) | QR/링크 (오픈 기간 내) |
| 교사 컨트롤 | 일시정지/건너뛰기/힌트 등 | 없음 |
| 이어하기 | 없음 | 있음 (UUID / 이메일+닉네임 기반) |
| 제출 횟수 | 1회 (자동) | 1회 (제출 확인 팝업) |
| 랭킹 공개 | 종료 즉시 | 마감 후 공개 (추후) |
| 정답 해설 | 즉시 | 공개 설정에 따라 |
| WebSocket | 필수 | 불필요 (REST만 사용) |

```
│  └─────────────────────────────────────────────┘    │
│                                                     │
│  ┌─────────────────────────────────────────────┐    │
│  │  최근 배포: 세트A · 라이브 · 완료율 85%      │    │
│  │  [리포트 보기 →]                            │    │
│  └─────────────────────────────────────────────┘    │
│                                                     │
│           [+ 새 세트지 만들기]                       │
└─────────────────────────────────────────────────────┘
```

| 기능 | 설명 |
|------|------|
| 최근 세트지 표시 | 최근 수정한 세트지 5개, 바로 배포 진입 가능 |
| 빠른 시작 | [+ 새 세트지 만들기] → 에디터 진입 |
| 최근 배포 현황 | 마지막 배포 완료율/날짜, 리포트 바로가기 |

| 빈 상태 | 처리 |
|---------|------|
| 세트지 0개 | "아직 만든 세트지가 없어요" + CTA |
| 배포 이력 없음 | "아직 배포한 게임이 없어요" |

### 공유 설정 다이얼로그

> 세트지 목록(S-02) 또는 에디터(S-03)에서 "커뮤니티에 공유" 선택 시 표시

```
+---------------------------------------------+
|  커뮤니티에 공유하기                    [x]   |
|                                             |
|  제목 *   [________________]                |
|  설명     [________________]                |
|  과목 *   [수학 v]                          |
|  학년 *   [중1 v]                           |
|  단원     [1단원 v]                         |
|  성취기준 [+ 추가] [9수01-01 x]             |
|  태그     [소수, 합성수____]                |
|                                             |
|  [취소]              [공유하기]              |
+---------------------------------------------+
```

**필드:**
- 제목 (필수): 기본값 = 원본 세트 제목
- 설명 (선택): 자유 텍스트
- 과목 (필수): 드롭다운 (수학, 영어, 과학, 사회 등)
- 학년 (필수): 드롭다운 (초1~고3)
- 단원 (선택): 과목/학년 선택 시 연동
- 성취기준 (선택): 검색형 멀티셀렉트 (`[9수01-01]` 형식)
- 태그 (선택): 쉼표 구분 자유 입력
