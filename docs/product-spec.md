## 1. 사용자 역할 정의

| 역할 | 설명 | 식별 방식 | 접근 경로 |
|------|------|-----------|-----------|
| **Host (교사)** | 세트지 제작·배포·리포트 조회 | 로그인 (UUID) | 대시보드 직접 접근 |
| **Member (멤버)** | 플랫폼 계정 보유 학생 | 로그인 (UUID) | QR·링크 스캔 후 로그인 |
| **Guest (게스트)** | 계정 없는 학생 | 이메일 + 닉네임 | QR·링크 스캔 후 정보 입력 |

**핵심 규칙**
- Host는 대시보드에만 직접 접근. Member·Guest는 QR·링크로만 진입.
- 게스트가 동일 이메일+닉네임으로 재접속 시 → 동일 인물로 이어받기 (기기·브라우저 무관).
- 게스트 이메일로 회원가입 시 → 기존 게스트 데이터 Member에게 연결 (회원가입 시점 자동 처리).

## 2. 화면 흐름 (Sitemap)

### 2-1. Host 대시보드

```
게임 진입
│
└── 홈 대시보드 (Host 전용)
      ├── [배포 전] 내 자료 영역
      │     ├── 세트지 목록
      │     ├── 배포 설정
      │     │     ├── 공통 설정
      │     │     │     └── 게임 유형 / 제한 시간 / 힌트 / 오답 재도전
      │     │     ├── [라이브] QR·링크 배포
      │     │     │     └── 배포 대상: 기존 그룹 선택 / 신규 그룹 생성 / QR 공개 배포
      │     │     └── [과제]
      │     │           ├── 오픈 기간 / 제출 횟수 / 정답 공개 시점
      │     │           ├── 멤버: 과제 알림 자동 이메일 발송
      │     │           └── 게스트: 알림 URL 복사 (호스트 수동 발송)
      │     └── 에디터 (자료 만들기)
      │
      ├── 그룹 관리
      │     ├── 그룹 목록 (생성·삭제)
      │     └── 그룹 상세
      │           ├── 멤버·게스트 목록 조회
      │           ├── 멤버 초대 (자동 이메일)
      │           ├── 게스트 초대 링크 복사
      │           ├── 멤버 제거
      │           └── 그룹에 배포 → 배포 설정 진입
      │
      └── [배포 후] 히스토리 & 리포트
            ├── 배포 히스토리
            │     ├── 배포 유형 표시 (라이브 / 과제)
            │     └── [과제] 마감일 D-day / 과제 마감 처리
            ├── [과제] 제출 현황
            │     ├── 제출완료 / 미제출 / 진행중 현황
            │     ├── 멤버별 제출 상태·점수 (UUID 기반)
            │     ├── 게스트별 제출 상태·점수 (이메일+닉네임 기반)
            │     ├── 미제출 멤버 필터
            │     └── 마감 연장 (추후)
            └── 리포트
                  ├── 문항별 분석
                  └── 학생별 분석
                        ├── 멤버: UUID 기반
                        └── 게스트: 이메일+닉네임 기반
```

### 2-2. 플레이 진행

```
플레이 진행
│
├── [라이브] QR 생성 & 대기화면 (Host 화면)
│     ├── Host: QR·링크 생성 및 화면 표시
│     ├── Member: QR 스캔 → 로그인 → 대기 입장
│     ├── Guest: QR 스캔 → 이메일+닉네임 입력 → 대기 입장
│     ├── Host: 입장한 멤버·게스트 닉네임 실시간 확인 (WebSocket)
│     ├── Host: 강퇴 / QR 재생성
│     └── Host: 게임 시작 (수동 트리거)
│
├── [과제] 과제 진입
│     ├── Member: 로그인 후 본인 과제 목록 확인 → 과제 선택
│     └── Guest: QR·링크 접속 → 이메일+닉네임 입력 → 바로 과제 진입
│                (게스트용 과제 목록 화면 없음)
│
├── 플레이 화면
│     ├── 문항 표시 / 카운트다운 / 정답·오답 피드백
│     ├── [라이브] Host 컨트롤 패널
│     │     ├── 일시정지 / 재개
│     │     ├── 문항 건너뛰기 / 시간 연장 / 힌트 공개
│     │     ├── 응답 현황 (N/N명 실시간)
│     │     └── 강제 종료
│     └── [과제]
│           ├── 이어하기 — Member: UUID 기반 세션 복원
│           ├── 이어하기 — Guest: 이메일+닉네임 기반 세션 복원
│           └── 제출 확인 팝업
│
└── 결과 화면 (라이브·과제 공통)
      ├── TOP 3 하이라이트
      ├── 전체 순위 목록 (멤버·게스트 구분 표시)
      ├── 내 순위 고정
      │     ├── Member: UUID 기반 본인 식별
      │     └── Guest: 닉네임 기반 본인 식별
      ├── 본인 점수·정답률
      ├── [라이브] 종료 즉시 랭킹 공개
      ├── [과제] 정답 해설 확인 (공개 설정에 따라)
      ├── [과제] 랭킹 마감 후 공개 (추후)
      └── 리플레이 / 종료 (Host 권한)
```

## 3. 기능 정의표

### 🏠 홈 대시보드
> Host(교사)만 접근. 내 자료·배포 현황을 한눈에 파악하는 허브.

| 기능 | 설명 | 사용자 | 엣지 케이스 |
|------|------|--------|------------|
| 최근 세트지 표시 | 최근 수정한 세트지 N개, 바로 배포 진입 가능 | Host | 세트지 0개일 때 empty state 처리 / 노출 개수 N 정책 결정 필요 |
| 빠른 시작 버튼 | [+ 새 세트지 만들기] → 에디터 진입 CTA | Host | |
| 최근 배포 현황 | 마지막 배포 완료율·날짜, 리포트 바로가기 | Host | 배포 이력 없으면 "아직 배포한 게임이 없어요" 안내 |

### 👥 그룹 관리

> Host가 그룹을 사전 구성하거나 자동 생성된 그룹을 관리. 멤버만 자동 이메일 발송. 게스트는 Host가 링크 직접 발송.

#### 그룹 목록

| 기능 | 설명 | 사용자 | 엣지 케이스 |
|------|------|--------|------------|
| 그룹 목록 조회 | 내가 Host인 그룹 전체. 그룹명·멤버 수·생성일 표시 | Host | |
| 그룹 직접 생성 | 그룹명 입력 후 사전 생성 | Host | |
| 그룹 삭제 | 그룹 삭제. 연결된 세션 데이터는 유지 | Host | 세션 이력 있는 그룹 삭제 시 "기존 데이터는 유지됩니다" 안내 |

#### 그룹 상세

| 기능 | 설명 | 사용자 | 엣지 케이스 |
|------|------|--------|------------|
| 멤버 목록 조회 | 그룹 내 멤버·게스트 목록. 유형·참여일 표시 | Host | 멤버(UUID) · 게스트(이메일+닉네임) 구분 표시 |
| 멤버 초대 (자동 이메일) | 멤버 추가 시 플랫폼이 초대 이메일 자동 발송. 내용: 그룹명·접속 링크 | Host | 이미 속한 멤버 재초대 시 중복 발송 방지 |
| 게스트 초대 링크 복사 | 게스트용 입장 링크 복사. Host가 수동 발송 | Host | 링크+안내 텍스트 함께 복사 제공 |
| 멤버 제거 | 특정 멤버·게스트와의 연결 차단 | Host | 기존 세션 데이터 유지 / 제거된 멤버의 QR 재입장 정책 필요 |
| 그룹에 배포 | 해당 그룹 기준으로 배포 설정 진입 | Host | 배포 설정의 "기존 그룹 선택"과 동일 진입점 |
