## S-13. 그룹 목록

**라우트** `/groups` | **역할** Host 전용

### 레이아웃

```
┌─────────────────────────────────────────────────────┐
│  그룹 목록                          [+ 새 그룹 만들기] │
├─────────────────────────────────────────────────────┤
│                                                     │
│  ┌─────────────────────────────────────────────┐    │
│  │  🏫 1학년 A반      15명   2026-01-03   [···] │    │
│  ├─────────────────────────────────────────────┤    │
│  │  🏫 2학년 수학반   8명    2026-01-10   [···] │    │
│  ├─────────────────────────────────────────────┤    │
│  │  🏫 방과후 영어    23명   2026-02-01   [···] │    │
│  └─────────────────────────────────────────────┘    │
│                                                     │
└─────────────────────────────────────────────────────┘

[···] 컨텍스트 메뉴:
  ├── 상세 보기
  ├── 이 그룹에 배포
  └── 삭제 (확인 Dialog)
```

### 상태 분기

| 상태 | UI 처리 |
|------|---------|
| 로딩 | 리스트 스켈레톤 |
| 그룹 0개 | EmptyState: "아직 만든 그룹이 없어요" + CTA |
| 삭제 시 세션 이력 존재 | Dialog: "기존 데이터는 유지됩니다. 그룹만 삭제할까요?" |

### 컴포넌트

| 컴포넌트 | 역할 | 파일 |
|----------|------|------|
| `GroupListItem` | 그룹 목록 행 (이름·멤버수·날짜·메뉴) | 인라인 구현 |
| `CreateGroupDialog` | 그룹명 입력 모달 | 인라인 구현 |

### API

```
GET    /api/groups              → Group[] (member_count 포함)
POST   /api/groups              → Group (그룹 생성)
DELETE /api/groups/[groupId]    → 204
```

## S-14. 그룹 상세

**라우트** `/groups/[groupId]` | **역할** Host 전용

### 레이아웃

```
┌─────────────────────────────────────────────────────┐
│  [← 그룹 목록]  1학년 A반  (15명)                   │
├─────────────────────────────────────────────────────┤
│                        [이 그룹에 배포 →]            │
├─────────────────────────────────────────────────────┤
│                                                     │
│  멤버 초대     [이메일 입력 ________________] [초대]  │
│  게스트 링크   [링크 복사 📋]                        │
│                                                     │
├─────────────────────────────────────────────────────┤
│  [전체 ▼]                           🔍 검색...      │
│                                                     │
│  ┌─────────────────────────────────────────────┐    │
│  │  홍길동   [Member]   참여일 2026-01-03  [제거] │    │
│  │  guest@ex [Guest]   참여일 2026-01-10  [제거] │    │
│  │  김철수   [Member]   참여일 2026-02-01  [제거] │    │
│  └─────────────────────────────────────────────┘    │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### 상태 분기

| 상태 | UI 처리 |
|------|---------|
| 로딩 | 리스트 스켈레톤 |
| 멤버 0명 | EmptyState: "아직 멤버가 없어요" |
| 이미 속한 멤버 초대 | Toast: "이미 그룹에 속한 멤버입니다" |
| 초대 성공 | Toast: "초대 이메일이 발송되었습니다" |
| 멤버 제거 | Confirm Dialog: "제거 후에도 기존 세션 데이터는 유지됩니다" |

### 컴포넌트

| 컴포넌트 | 역할 | 파일 |
|----------|------|------|
| `MemberRow` | 멤버/게스트 행 (유형·날짜·제거) | 인라인 구현 |
| `InviteMemberInput` | 이메일 입력 + 초대 버튼 | 인라인 구현 |

### API

```
GET    /api/groups/[groupId]                      → Group + GroupMember[]
POST   /api/groups/[groupId]/invite               → 204 (이메일 발송)
DELETE /api/groups/[groupId]/members/[memberId]   → 204
GET    /api/groups/[groupId]/guest-link           → { link: string }
```

### 상태 분기

| 상태 | UI 처리 |
|------|---------|

### 데이터 구조

```typescript
interface QuestionReport {
  question: Question
  total_participants: number
  correct_count: number
  correct_rate: number            // 0~1
  avg_response_time_sec: number
  distribution: Record<string, number>  // "1": 3, "2": 11, ...
  top_wrong_answers?: string[]    // 단답형 전용
}
```

### API

```
GET /api/sessions/[sessionId]/report/questions   → QuestionReport[]
```

### 레이아웃

```
┌─────────────────────────────────────────────────────┐
│  [← 히스토리]  수학 1단원 평가  |  학생별 분석       │
│                                  [← 문항별 분석]     │
├─────────────────────────────────────────────────────┤
│  🔍 검색...           [미참여만 보기]                │
│                                                     │
│  ┌─────────────────────────────────────────────┐    │
│  │  순위  이름      유형     점수    정답  시간  │    │
│  ├─────────────────────────────────────────────┤    │
│  │  1위  홍길동  [Member]  950점  10/12  1:05  │    │
│  │  2위  김철수  [Guest]   900점  9/12   1:23  │    │
│  │  ···                                        │    │
│  │  —    이영희  [Member]  -      미참여  -     │    │
│  └─────────────────────────────────────────────┘    │
│                                                     │
│  클릭 시 → 학생 상세 (문항별 정오답 내역 펼침)       │
│  ┌─────────────────────────────────────────────┐    │
│  │  홍길동 상세                                │    │
│  │  Q1 광합성... ✅ ② 엽록체  8.2초           │    │
│  │  Q2 세포...   ✅ ① 세포    5.1초           │    │
│  │  Q3 핵...     ❌ ③ 핵 → 정답: ② 엽록소    │    │
│  └─────────────────────────────────────────────┘    │
└─────────────────────────────────────────────────────┘
```

### 상태 분기

| 상태 | UI 처리 |
|------|---------|
| 로딩 | 테이블 스켈레톤 |
| 참여자 0명 | "데이터가 없습니다" EmptyState |
| 미참여 학생 | 순위 없음, "-" 표시 |
| 닉네임 미허용 세션 | "닉네임 미허용 설정으로 학생별 분석을 제공할 수 없습니다" 안내 |

### 데이터 구조

```typescript
interface StudentReport {
  result: ParticipantResult         // 순위·점수·응답시간
  responses: ResponseEvent[]        // 문항별 응답 내역
  questions: Question[]             // 문항 정보 (정답 표시용)
}
```

### API

```
GET /api/sessions/[sessionId]/report/students          → StudentReport[]
GET /api/sessions/[sessionId]/report/students/[resultId] → StudentReport (상세)
```
